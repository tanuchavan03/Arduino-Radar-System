# Arduino-Radar-System
This project uses an Arduino Uno, HC-SR04 Ultrasonic Sensor, and Servo Motor to detect objects and measure their distance. The sensor scans the surrounding area by rotating the servo motor, while the Processing application visualizes the detected objects on a radar-like interface in real time.
# Arduino Radar System

## Overview
This project is an Arduino-based radar system that uses an HC-SR04 Ultrasonic Sensor and a Servo Motor to detect objects and measure their distance.

## Components Used
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- Servo Motor
- Jumper Wires
- Processing IDE

## Features
- Real-time object detection
- Distance measurement
- Radar-style visualization
- Servo motor scanning

## Technologies Used
- Arduino IDE
- Processing IDE
- Embedded C/C++

## Project Type
Academic Team Project
